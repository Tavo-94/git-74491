# clase 03 - git desarrollo colaborativo

## Repaso

### Estrategia de ramas basica
![estructuras-ramas](_ref/estrategiaBranches.webp)

## creando una rama

```sh
git branch <nombre-rama>
```

## listar ramas

```sh
git branch
```

## cambiar de ramas

```sh
git switch <nombre-rama>
```

## toogle entre las 2 ultimas ramas visitadas

```sh
git switch -
```
